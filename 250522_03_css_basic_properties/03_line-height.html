글자행간 지정하기

문단의 행간을 지정해야할 경우, line-height 속성을 사용합니다. 행간을 지정할 때 글자의 크기보다 크게 지정해 주어야 합니다. line-height 값을 입력할 때 숫자로 입력하여야합니다. 행간지정은 폰트의 1.2배 ~ 1.7배 정도의 크기가 적정합니다.
